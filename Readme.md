
# Описание программы:

Программа построена на графическом интерфейсе easygui

Имеет следующие функции
    * Показать все контакты
    * Добавить контакт
    * Удалить контакт
    * Сохранить контакты в зыписную книжку
    * Загрузить контакты из записной книжки
    * Изменение контакта
        ** Изменение имени
        ** Изменение телефона
        ** Изменение даты рождения
        ** Изменение Email
